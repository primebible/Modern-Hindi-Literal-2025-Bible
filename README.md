### कौन-सा PDF चुनें? (Which PDF should I choose?)

| नाम में लिखा है / Name says | किसके लिए / Best for |
|---|---|
| **Standard** | रोज़ पढ़ने के लिए — दो कॉलम / everyday reading, two columns |
| **Economy** | सबसे सस्ती छपाई (A5 कागज़) / cheapest printing (A5 paper) |
| **Premium** | जिल्दसाज़ी और उपहार (6×9 इंच) / bookbinding & gifts (6×9 in) |
| **Mobile** | मोबाइल पर पढ़ने के लिए / reading on a phone |
| **-LargePrint** | बड़े अक्षर — कमज़ोर नज़र के लिए / big letters for weak eyesight |

### कैसे छापें (How to print)

1. ऊपर की तालिका से PDF चुनें और **Download** दबाएँ। *Pick a PDF above and press Download.*
2. छापते समय **100% size** चुनें ("fit to page" नहीं)। *Print at 100% size (not "fit to page").*
3. कागज़: Economy = A5 · Standard ≈ B5 · Premium = 6×9 इंच। *Paper sizes per edition.*
4. जितनी चाहें उतनी प्रतियाँ छापें और बाँटें। 🙏 *Print and share freely.*

हर PDF के दूसरे पन्ने पर **पाठ संस्करण (text version)** और **तारीख़** लिखी है — नई तारीख़ = नया संस्करण।
*Every PDF's second page shows the text version and compile date — a newer date means a newer printing.*

## 🎯 Our Goal: A Bible for Everyone (हमारा लक्ष्य)

भारत में लाखों लोग बाइबिल पढ़ना चाहते हैं, लेकिन एक बड़ी बाधा का सामना करते हैं: **भाषा।**

सबसे आम हिन्दी बाइबिल (पवित्र बाइबिल, OV) 1883 में लिखी गई थी। वह भाषा 140+ साल पुरानी, अर्काइक और पढ़ने में कठिन है।

**हमारा लक्ष्य सरल है:** परमेश्वर के वचन को एक ऐसी स्पष्ट, सुंदर और आधुनिक हिन्दी में प्रस्तुत करना, जिसे कोई भी, किसी भी शैक्षिक पृष्ठभूमि का व्यक्ति, आसानी से पढ़ और समझ सके।

## 🤝 How We Are Doing It (हमारा तरीका)

**यह एक मूल अनुवाद है, कोई कॉपी नहीं है।**

हमारा अनुवाद कार्य सार्वजनिक डोमेन (public domain) स्रोतों पर आधारित है, जिसमें मूल यूनानी पांडुलिपियाँ और वर्ल्ड इंग्लिश बाइबिल (WEB) शामिल हैं। यह हमें एक ऐसा अनुवाद बनाने की स्वतंत्रता देता है जो:

- **सटीक (Accurate):** मूल अर्थ के प्रति वफादार।
- **आधुनिक (Modern):** पुरानी, भ्रामक भाषा के बजाय आज की हिन्दी का उपयोग करना।
- **सुलभ (Accessible):** सभी के लिए पढ़ने में आसान।

## 🗂️ How It Is Organized (संरचना)

प्रत्येक पुस्तक का अपना फ़ोल्डर होता है। हर फ़ोल्डर में:

- `1.yaml`, `2.yaml`, … — अनुवाद का पाठ, एक फ़ाइल प्रति अध्याय (अनुवादक इन्हीं को संपादित करते हैं)
- `<Book>-Standard.pdf`, … — छपाई के लिए तैयार PDF, हर संस्करण में — **पाठ बदलते ही अपने-आप फिर से बनते हैं**

Each book has its own folder containing the chapter YAML files (what
translators edit) and the print-ready PDFs in every edition, **rebuilt
automatically whenever the text changes** (via GitHub Actions).

- [`Complete-Bible/`](Complete-Bible/) — अब तक का पूरा अनुवाद एक ही PDF में। *Everything translated so far in one PDF.*
- [`compiler/`](compiler/) — PDF बनाने वाला टूल (तकनीकी)। *The tool that builds the PDFs — see [compiler/README.md](compiler/README.md). Translators never need to touch it.*

## 🙏 How to Contribute (योगदान कैसे दें)

यह एक सामुदायिक प्रयास है! आप मौजूदा अनुवादों की समीक्षा करके, त्रुटियों को ठीक करके, या नए अध्यायों का अनुवाद करके मदद कर सकते हैं।

*This is a community effort! You can help by reviewing existing translations, fixing errors, or translating new chapters.*

आरंभ करने के लिए:

1. इस रिपॉजिटरी को **fork** करें। *(Fork this repository.)*
2. अपने परिवर्तन करें। *(Make your changes — chapter YAML lives right in each book's folder.)*
3. एक **Pull Request** सबमिट करें। *(Submit a Pull Request. The PDFs rebuild automatically after merge.)*

आप कोई प्रश्न या सुझाव साझा करने के लिए एक [issue](../../issues) भी खोल सकते हैं। *You can also open an [issue](../../issues) to ask a question or share a suggestion.*

⭐ यदि यह परियोजना आपके लिए उपयोगी है, तो कृपया इसे **star** करें और दूसरों के साथ साझा करें। *If this project helps you, please star it and share it with others.*
